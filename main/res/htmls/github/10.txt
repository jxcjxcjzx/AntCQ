    <h3>
      <span class="octicon octicon-book"></span>
      README.md
    </h3>

    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><p>Welcome to Echo Nest Remix: The (JavaScript) Internet Synthesizer</p>

<p>Echo Nest Remix is <strong>the Internet Synthesizer.</strong> 
Make amazing things from music, automatically.  Turn any music Javascript code, cut it up, move through it, and control it.</p>

<p>It is based on code from the mighty <a href="http://www.infinitejuke.com/">Infinite Jukebox</a>, 
and makes use of the WebAudio API to cut, queue, and remix music.</p>

<h2><a id="user-content-getting-started" class="anchor" href="#getting-started" aria-hidden="true"><span class="octicon octicon-link"></span></a>Getting Started</h2>

<p>We've made a shiny new page for working with Remix: <a href="http://echonest.github.com/remix/javascript.html">check it out.</a> 
This is a new project, so things may change very rapidly - let us know if you have any problems!</p>

<p>-<a href="https://camo.githubusercontent.com/c7a3810cd59b15375246e5468b46cdecd18edbb9/687474703a2f2f692e696d6775722e636f6d2f57574c596f2e676966" target="_blank"><img src="https://camo.githubusercontent.com/c7a3810cd59b15375246e5468b46cdecd18edbb9/687474703a2f2f692e696d6775722e636f6d2f57574c596f2e676966" alt="alt text" title="Head-nodding-cat.js" data-canonical-src="http://i.imgur.com/WWLYo.gif" style="max-width:100%;"></a></p>

<p>(If you're looking for the original, Python version of remix, go <a href="https://github.com/echonest/remix">here.</a>)</p>
