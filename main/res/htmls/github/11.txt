<h2><a id="user-content-generating-base64-soundfonts" class="anchor" href="#generating-base64-soundfonts" aria-hidden="true"><span class="octicon octicon-link"></span></a>Generating Base64 Soundfonts</h2>

<p>There is two generators for MIDI.js soundfonts:</p>

<ul>
<li>NodeJS package for creating soundfonts from WAV files - by Patrick Wolleb</li>
<li>Ruby package for creating soundsfonts from SF2 files - by Mohit Muthanna</li>
</ul>

<p>To dive in quickly Benjamin Gleitzman has created a package of <a href="https://github.com/gleitz/midi-js-soundfonts">pre-rendered sound fonts</a>.</p>

    instruments<span class="pl-k">:</span> [ <span class="pl-s"><span class="pl-pds">"</span>acoustic_grand_piano<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>acoustic_guitar_nylon<span class="pl-pds">"</span></span> ], <span class="pl-c">// or multiple instruments</span>
