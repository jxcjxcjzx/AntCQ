<p>Backbone supplies structure to JavaScript-heavy applications by providing models with key-value binding and custom events, collections with a rich API of enumerable functions, views with declarative event handling, and connects it all to your existing application over a RESTful JSON interface.</p>

<p>For Docs, License, Tests, pre-packed downloads, and everything else, really, see:
<a href="http://backbonejs.org">http://backbonejs.org</a></p>

<p>To suggest a feature, report a bug, or general discussion:
<a href="https://github.com/jashkenas/backbone/issues">https://github.com/jashkenas/backbone/issues</a></p>

<a href="https://github.com/documentcloud">https://github.com/documentcloud</a></p>
