    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-flat-ui-free" class="anchor" href="#flat-ui-free" aria-hidden="true"><span class="octicon octicon-link"></span></a><a href="http://designmodo.github.io/Flat-UI/">Flat UI Free</a></h1>

<p>Flat UI is a beautiful theme for <a href="http://getbootstrap.com">Bootstrap</a>. We have redesigned many of its components to look flat in every pixel.</p>

<p>To get started, check out <a href="http://designmodo.github.io/Flat-UI/docs/getting-started.html">getting started page</a> or follow instructions below.</p>

<h2><a id="user-content-links" class="anchor" href="#links" aria-hidden="true"><span class="octicon octicon-link"></span></a>Links:</h2>

<ul>
<li><a href="http://designmodo.github.io/Flat-UI/">Demo Page</a></li>
<li><a href="http://designmodo.com/flat-free/">Download PSD</a> (for designers)</li>
<li><a href="http://designmodo.com/flat/">Flat UI Pro 1.3.2</a> (from $39)</li>
</ul>

<h2><a id="user-content-quick-start" class="anchor" href="#quick-start" aria-hidden="true"><span class="octicon octicon-link"></span></a>Quick start</h2>

<p>Three quick start options are available:</p>

<ul>
<li><a href="https://github.com/designmodo/Flat-UI/archive/2.2.2.zip">Download the latest release</a>.</li>
<li>Clone the repo: <code>git clone https://github.com/designmodo/Flat-UI.git</code>.</li>
<li>Install with <a href="http://bower.io">Bower</a>: <code>bower install flat-ui</code>.</li>
</ul>

<p>Run <code>npm install</code> and <code>bower install</code> to install dependencies.</p>

<p>Read the <a href="http://designmodo.github.io/Flat-UI/docs/getting-started.html">getting started page</a> for information on the framework contents.</p>

<p>Examples: <a href="https://github.com/designmodo/Flat-UI/tree/master/docs/examples">https://github.com/designmodo/Flat-UI/tree/master/docs/examples</a>.</p>
